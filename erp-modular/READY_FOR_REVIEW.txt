2. ออกแบบ Data Model ด้วย Pydantic v2 + SQLModel:

   - App: Mini App ที่เชื่อมต่อผ่าน API Gateway
3. สร้าง FastAPI CRUD สำหรับ Module/Template
4. สร้าง Abstract Base Class สำหรับ Plugin Interface
